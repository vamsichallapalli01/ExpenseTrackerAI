# Expense Tracker AI

A modern personal finance management application built with Django.

Features include:

- Income Tracking
- Expense Tracking
- Budget Management
- Analytics Dashboard
- Charts
- AI Spending Insights
- Recurring Expenses
- Notifications
- User Profiles
- Dark Mode

## Features

### Financial Management

- Add Income
- Add Expenses
- Monthly Budget Tracking
- Savings Calculation

### Analytics

- Expense Breakdown Chart
- Monthly Trend Chart
- Spending Analysis

### Smart Features

- AI Insights
- Notifications
- Budget Alerts

### User Features

- Registration
- Login
- Profile Management
- Dark Mode

## Installation

Clone repository:

```bash
git clone https://github.com/yourusername/ExpenseTrackerAI.git
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run migrations:

```bash
python manage.py migrate
```

Start server:

```bash
python manage.py runserver
```
